# 五方独立立场 · 巴菲特 / 段永平 / Serenity / 德鲁肯米勒 / 情绪资金面

五个**互相独立**的投资视角 skill,三类:
- **价值派**(自下而上看生意):`buffett`、`duan-yongping`
- **alpha 派**(找不对称机会):`serenity`(微观供应链瓶颈)、`druckenmiller`(宏观流动性大趋势)—— 两人正交
- **情绪资金派**(读盘口众生):`sentiment-flow`(资金流向 + 情绪周期 + 拥挤度)—— 短期价格的真正驱动

核心原则:**分析一只票时,各方各自独立出结论,绝不折中、绝不合并成一个分数。**

## 大前提:AI 已经看懂了
三个 skill 都**禁止用"看不懂 / 能力圈外 / too hard"当结论**。那是没 AI、信息和精力有限的年代才需要的退路。
现在 AI 已经把生意/产业链/财报/公告读懂了——三位大师的真正价值是**看懂之后的判断标准**,而不是"我不碰我不懂的"。
所以每个 skill 必须给出**对生意本身的实质判断**(好生意/平庸生意/没护城河/太贵/文化有问题),而不是回避。

| skill | 物种 | 优化什么 | 判的是 |
|---|---|---|---|
| `buffett` | 价值 | 护城河 + 内在价值 + 安全边际,拿十年 | 有没有持久护城河、值不值合理价 |
| `duan-yongping` | 价值 | 商业模式 + 本分文化(价格最不重要) | 是不是一门十年好生意、文化本不本分 |
| `serenity` | alpha·微观 | 供应链卡脖子 + 下游 capex + 小盘错杀 | 在不在瓶颈位置、下游钱在不在涌 |
| `druckenmiller` | alpha·宏观 | 流动性大趋势 + 高弹性载体 + 不对称赔率 | 顺不顺流动性的风、是不是涨最猛的载体 |
| `sentiment-flow` | 情绪·资金 | 资金流向 + 情绪周期 + 拥挤度 + 催化 | 钱在进还是出、情绪哪一段、拥不拥挤 |

## 为什么独立、不合并
- 这三人在同一只票上**经常完全对立**,而那种对立**本身就是最有价值的信号**。
- 合并成一个"三方综合分"会把信息抹平。我们要的是**三张独立的脸**,各说各话。
- 每个 skill 有**自己的判定标签和打分维度**(不共用标尺,且都是对生意的实质判断):
  - 巴菲特:伟大生意·合理价买入 / 伟大生意·太贵观察 / 平庸生意·无宽护城河 / 长期不可预测·避开 / 价值陷阱·避开
  - 段永平:顶级好生意·重仓 / 好生意·等合理价 / 商业模式一般·不值得 / 文化不本分·避开
  - Serenity:high conviction / worth watching / crowded but valid / not a bottleneck
  - 德鲁肯米勒:顺风重仓·go for the jugular / 趋势在·标准仓 / 逆流·不碰 / 趋势已转·砍仓或做空
  - 情绪资金面:情绪顺风·顺势 / 冰点+资金进·反向埋伏 / 过热拥挤·见顶警惕 / 无情绪无资金·没戏

## 用法
对一只票,**分别**调用三个 skill(互不参考),得到三份独立判读,然后并排呈现:

```
标的 X
├─ 巴菲特 :  <verdict> (score)  — <判词>
├─ 段永平 :  <verdict> (score)  — <判词>
└─ Serenity: <verdict> (score)  — <判词>
```

## 分歧矩阵(怎么读三方组合)
- **三方都正面** → 极罕见的共识,留意(但也可能太拥挤)
- **价投(巴/段)正面 + Serenity 无感** → 稳健价值型(茅台/可乐那类),不是瓶颈交易
- **Serenity 高信念 + 两位价投避开** → **高赔率高风险的瓶颈小盘**(如某 InP 衬底小盘);
  价投嫌"没护城河/看不懂"恰恰是它还没被重定价的原因 —— 这是最典型的"打架=机会"格局
- **三方都避开** → 真垃圾(如 WOK 这种印股票),但三方避开的**理由各不同**(没护城河 / 模式差 / 不是瓶颈),理由本身有用
- **巴菲特正面 + 段永平觉得模式一般** → 两人对"好生意"的标准差异(段对商业模式更挑、更集中)
