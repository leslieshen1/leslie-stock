## 总则：5 条不可妥协的信条

### 信条 1：买股票就是买公司（巴菲特）
我们买的不是代码，不是 K 线，是企业的部分所有权。
衡量一只股票贵不贵，看它对应的企业**值不值这个价**。

### 信条 2：做对的事，把对的事做对（段永平）
- **做对的事** = stop doing list（不做错事比做对事更重要）
- **把对的事做对** = 在能力圈内做深做精
- 段永平："如果一件事错了，再怎么努力把它做'好'，也是错的"

### 信条 3：以"永远持有"为默认买入心态
- 买入前问自己：如果未来 10 年这只股票不能卖，我还买吗？
- 买入即是合伙人 —— 你不会每周问一次"要不要换个合伙人"
- 频繁交易是负和游戏（手续费 + 错过复利 + 情绪损耗）

### 信条 4：不预测
我们不预测：
- 大盘点位
- 利率走向
- 下季度业绩具体数字
- 短期资金流向
- 政策细节

我们只判断：**"这门生意 10 年后大概率怎么样？"**

### 信条 5：集中 + 长期 + 慢富
- 段永平一生重仓不超过 10 只
- 巴菲特："把鸡蛋放在一个篮子里，然后看好它"
- 巴菲特年化 ~20% 复利 50 年 → 财富积累的本质是**复利时间**，不是单次暴利
- "慢即快"

## 第一部分：六维度评估框架

每一只候选股票，都按以下 6 个维度逐项打分。**任何一项不过关，一票否决。**

### 维度 1：商业模式（Business Model）

**核心问题**：这是一门好生意吗？10 年后还在赚钱吗？

#### 好生意的特征
- 长坡厚雪：行业空间大、增长持久（可口可乐百年增长、白酒消费长青）
- 复用性强：今天的努力对明天有用（品牌积累、用户数据、技术资产）
- 资本支出小：赚的钱不需要全部再投入（茅台 vs 钢铁厂）
- 定价权：能跟随通胀提价（苹果、茅台、爱马仕）
- 高 ROIC + 高自由现金流（钱在创造钱）

#### 差生意的特征
- 同质化竞争（钢铁、航空、面板、光伏组件最终阶段）
- 重资产 + 强周期性（化工、航运、地产、有色周期顶部）
- 技术快速迭代 + 必须持续大投入（部分半导体设备）
- 监管 / 政策悬而未决（部分医药集采、互联网监管期）

#### 段永平判断标准
> "我从不投差行业里的好公司，宁可投好行业里的中等公司。"

#### 行业评分参考（你的能力圈范围）
- **AI 上下游**：算力基础设施 / 应用层差异巨大；龙头有"网络效应+规模"双重优势，二三线没有
- **能源 / 材料 / 资源**：周期性强，多数是"差生意"；但稀缺资源（铜、稀土）+ 龙头成本优势 = 例外
- **医药生物**：创新药"赢家通吃" + CXO"卖铲子"是好生意；仿制药受集采压价
- **互联网**：平台型公司（腾讯、美团、拼多多）是顶级好生意；垂类应用看护城河

### 维度 2：护城河（Moat）

**核心问题**：这家公司有什么，是别人 5 年内学不会、买不到、抢不走的？

#### 护城河的 5 种类型（巴菲特+晨星分类）

| 类型 | 定义 | 代表案例 |
|---|---|---|
| **品牌** | 消费者愿为相同产品付溢价 | 茅台、苹果、可口可乐、爱马仕 |
| **网络效应** | 用户越多，对每个用户越有价值 | 腾讯（微信）、淘宝、滴滴 |
| **转换成本** | 客户换供应商代价大 | 云服务、ERP、银行账户、企业 SaaS |
| **成本优势** | 单位成本结构性领先 | 神华煤炭、紫金铜矿、宁德产能 |
| **特许牌照** | 监管壁垒 | 银行、保险、白酒、烟草、稀土配额 |

#### 护城河的动态性
- 护城河会**侵蚀**（柯达、诺基亚、Yahoo）
- 每年都要重新审视，不能假设永恒
- 技术驱动的护城河 < 品牌 / 网络效应护城河
- 警惕"伪护城河"：补贴换市占率、广告砸出来的品牌、靠政策吃饭

#### 检验问题
- 过去 5 年护城河变宽还是变窄？
- 新进入者要花多少钱、多少年才能复制？
- 如果竞争对手有 100 亿美金，能打垮它吗？

### 维度 3：管理层（Management）

**核心问题**：老板靠谱吗？言行一致吗？把股东当合伙人吗？

#### 段永平 / 巴菲特看管理层的 4 个角度

1. **诚信**
   - 财报干净、年报实话
   - 不放卫星、不画大饼
   - 出错敢承认（段永平退出步步高某些业务）

2. **理性**：资本配置是否合理
   - 多余现金：分红 / 回购 / 投资 / 扩产？
   - 并购：贵不贵？战略对路吗？
   - 扩产：行业周期位置？

3. **能力圈意识**
   - 不乱投资、不跨界、不追风口
   - 段永平：步步高分拆 OPPO / vivo / 步步高教育，每家专注

4. **股东导向**
   - 把股东当合伙人而非工具人
   - 不大额减持、不利益输送
   - 信息披露透明、年报愿意写真话

#### A 股 / 港股特别警报

🚨 一票否决信号：
- 大股东 / 实控人**质押率 > 50%**（爆仓风险）
- 大股东 / 高管**连续大额减持**（信任崩塌）
- **关联交易占比异常高**（利益输送嫌疑）
- 财报频繁**业绩"洗澡"** / 商誉巨额减值
- 实控人有**违法 / 失信记录**
- 频繁**变更会计师事务所** / 财务总监

#### 国企 vs 民企的差异看法
- 国企：看政策周期 + CEO 任期；分红稳定但增长慢
- 民企：核心是**老板个人**（丁磊、张一鸣、王传福、宁德曾毓群）

### 维度 4：财务质量（Financial Quality）

**核心问题**：这家公司每年到底赚多少真钱？利润是不是真的？

#### 财务红线（卡死的指标）

| 指标 | 标准 | 警示线 | 备注 |
|---|---|---|---|
| **ROE（净资产收益率）** | 长期 > 15%，最好稳定 > 20% | 单年 < 10% 警告 | 巴菲特核心指标 |
| **ROIC（投入资本回报率）** | > 10%（资本成本之上） | < 8% 警告 | 比 ROE 更纯粹 |
| **毛利率** | 稳定或上升 | 连续下降 2 年 | 反映护城河变化 |
| **净利率** | 稳定或上升 | 同上 | 同上 |
| **自由现金流（FCF）** | 连续 5 年正 | 单年负不一定差，连负警告 | "真金白银" |
| **FCF / 净利润** | > 70% | < 50% 警告 | 利润质量 |
| **经营现金流 / 净利润** | > 1 | < 0.8 警告 | 收入是否变现金 |
| **资产负债率** | < 60%（金融除外） | > 70% 警告 | 高杠杆风险 |
| **应收账款增速** | ≤ 营收增速 | > 营收增速 50% 警告 | 是否塞货 |
| **存货 / 营收** | 稳定 | 大幅上升 | 是否积压 |
| **商誉 / 净资产** | < 30% | > 50% 警告 | 减值地雷 |
| **股息率**（成熟期公司） | > 2% | 一直不分红的成熟公司警告 | 现金真伪检验 |

#### 段永平的财务质量三问
1. 每年到底赚多少真钱？→ **自由现金流**
2. 这些钱怎么花的？→ **资本配置**
3. 不再扩张的话，每年能分多少给股东？→ **分红能力 + 回购能力**

#### 周期股 / 金融股的特殊处理
- 周期股看 PB + 行业景气位置，不能只看 PE
- 银行 / 保险看 ROE + 不良率 + 拨备覆盖率
- 房地产看销售回款 / 存货周转 / 净负债率

### 维度 5：估值与安全边际（Valuation & Margin of Safety）

**核心理念**：
- 估值不是预测，是检验"是否值得买"
- 不需要精确，只需要"明显便宜"或"不算贵"
- 段永平："只有看不见的便宜，没有看不见的贵"

#### 4 种简化估值法

##### A. 历史分位法（最常用）
- PE / PB / PS / 股息率，看当前在 5 年 / 10 年的百分位
- < 30% 分位：偏便宜
- 30%-70%：合理区间
- > 70%：偏贵
- **使用**：用于成熟稳定公司（消费、医药龙头、公用事业）

##### B. 股票收益率法（巴菲特常用）
- 股票收益率 = 自由现金流 / 市值
- 对比 10 年期国债收益率
- 比国债高 50%+ 就值得看
- **使用**：用于现金流稳定的公司

##### C. 十年回本法（段永平思维）
- 假设公司不分红、不增长，多少年能用净利润还本？
- < 15 年：可接受
- < 10 年：有吸引力
- < 7 年：明显便宜
- **使用**：快速过滤工具

##### D. DCF 思维（不需精确）
- 10 年后这家公司净利润大概多少？
- 给一个合理 PE，倒推今天合理市值
- 当前市值 vs 倒推市值，至少 30% 折扣才买
- **使用**：复杂场景，但不要陷入精确数字陷阱

#### 估值锚定的"YES/NO"清单
- ✅ 接受：PE、PB、股息率、FCF / 市值、十年回本期
- ❌ 拒绝："故事估值"（PS for 持续亏损公司、用户数、GMV、想象空间）
- ⚠️ 谨慎：成长股的 PEG（增长预测本身就不可靠）

#### 安全边际原则
- **内在价值估算 - 当前市值 ≥ 30%**：明显低估，买入
- **估值偏贵但生意极好**（茅台、苹果某些阶段）：可分批 + 长期持有
- **估值便宜但生意一般**（部分周期股）：仅在极端便宜时博弈，做好快进快出准备
- **估值贵 + 生意一般**：永远不碰

#### "好公司也要好价格"
段永平特别强调：
> "我等苹果等了好多年，等到合适的价格才买。"
> 好公司不等于任何价格都买。

### 维度 6：能力圈（Circle of Competence）

**核心原则**：**宁可错过，不要做错**

#### 你的能力圈（Leslie，2026-05 起点）

🎯 **核心圈 / 学习圈（待你校准）**：
- AI 上下游（算力 / 大模型 / 应用 / 终端）
- 能源 / 材料 / 资源（新能源 / 锂电 / 有色 / 传统能源）
- 医药生物（创新药 / CXO / 器械 / 中药）
- 互联网（重在港股：腾讯、美团、阿里、拼多多、网易、B 站等）

#### 能力圈分层管理（核心机制）

| 层级 | 标准 | 操作 |
|---|---|---|
| **核心圈** | 用过产品 + 跟踪 3 年+ + 跌 50% 敢加仓 | 可重仓（单只最高 30%） |
| **学习圈** | 感兴趣但还不深入 | 跟踪不买，研究 6-12 个月再说 |
| **圈外** | 完全没接触过 | 自动过滤，不浪费时间 |

#### 能力圈的 4 个自检问题
1. 我能在 5 分钟内向门外汉讲清这门生意吗？
2. 我能解释这家公司今年为啥赚 X 元、明年大概多少吗？
3. 我知道这家公司面临的 3 个最大风险吗？
4. 一年不看股价，我会担心吗？（答 No 说明真的懂）

#### 能力圈扩张原则
- 不是"我看了几篇研报"就懂了
- 至少跟踪 4 个季度 + 看至少 3 份年报 + 用过它的产品（如果可能）
- 跟一个行业的人聊过（专业内人士）
- 经历过一次完整的财报季 + 一次股价大波动

## 第二部分：段永平的 Stop Doing List

> "不做什么"比"做什么"更重要。
> —— 段永平

**绝对不做的事**（一旦违反，即刻自查）：

| 不做 | 理由 | 例外 |
|---|---|---|
| **不做空** | 损失无限，胜率低 | 无 |
| **不用杠杆 / 不借钱投资** | 杠杆让你在最低点被强平 | 无 |
| **不预测大盘 / 不择时** | 没人能持续做对 | 无 |
| **不看 K 线 / 不用技术分析** | 价格波动不反映价值 | 无 |
| **不打听消息 / 不听小道** | 你听到时已经不值钱 | 无 |
| **不追热点 / 不参与 IPO** | IPO 是卖方主导，结构性吃亏 | 无 |
| **不参与题材炒作** | 妖股 95% 最后归零 | 无 |
| **不为分散而分散** | 持有不懂的就是赌博 | 无 |
| **不频繁交易** | 摩擦成本 + 错过复利 + 情绪损耗 | 无 |
| **不碰看不懂的** | 圈外 = 赌 | 无 |
| **不在情绪上头时操作** | 暴跌想割、暴涨想追 = 反向信号 | 至少冷静 24 小时 |
| **不碰高估的好公司** | 好公司也要好价格 | 极少数（如苹果某阶段）可分批小仓位 |
| **不碰便宜的烂公司** | 价值陷阱 | 无 |

## 第四部分：卖出三条件（段永平名言）

> 卖出只有三个理由：
> 1. 当初买错了（判断错了）
> 2. 基本面变坏了
> 3. 找到明显更好的
>
> —— 段永平

### 条件 1：当初买错了
- 复盘买入备忘录的"核心逻辑"
- 假设根本不成立 → 立即承认 + 卖出
- ⚠️ 警惕：不要因为已亏损而拒绝承认错误（沉没成本）
- ⚠️ 警惕：不要因为已盈利而合理化错误买入

### 条件 2：基本面变坏
- 商业模式 / 护城河 / 管理层 / 财务质量任一**结构性**恶化
- 例子：
  - 商誉巨额减值
  - 关联交易激增（利益输送）
  - 核心管理层不和 / 出走
  - 主营业务市占率持续下滑
  - 政策结构性逆转（教培、互联网监管初期）
- ⚠️ 区分：**周期性**下行（继续持有 / 加仓）vs **结构性**下行（卖出）

### 条件 3：找到明显更好的
- 当前持仓预期年化 15%，新机会预期 25%+
- "明显更好" = 预期回报至少高 50%+
- ⚠️ 调仓也要克制：不为微小优势频繁换
- ⚠️ 验证："新机会"是否真在能力圈内，还是 FOMO？

### **不是**卖出的理由
- ❌ 股价涨了很多（涨多了也别轻易卖好公司）
- ❌ 短期跌了很多（跌了反而审视是否加仓）
- ❌ 市场普遍看空 / 看多
- ❌ "感觉要回调"
- ❌ 别人都在卖 / 都在买
- ❌ 分析师下调评级
- ❌ 短期业绩不及预期（要看是趋势还是噪音）

## 第五部分：仓位管理

### 仓位分配原则

```
- 核心仓位（5-10 只）：80-90% 资金
- 学习仓位（试水）：5-10% 资金，单只 < 3%
- 现金 / 货币基金 / 短债：10-20%（巴菲特"等待挥棒"）
- 不打满仓：留 10-20% 应对暴跌机会
```

### 现金的价值
- 巴菲特："现金是一份看涨期权 —— 等待大跌的最佳武器"
- 段永平："满仓的人没有勇气加仓"
- 长期保持 10-20% 现金不是浪费，是机会准备金

### 加仓 / 减仓 / 不动的信号

| 信号 | 操作 |
|---|---|
| 股价下跌 + 基本面无变化 + 估值更便宜 | **加仓** |
| 估值极端高估（PE 95%+ 分位）+ 生意没显著变好 | 考虑**减仓**（不必全清） |
| 股价上涨 + 基本面正常 | **什么都不做**（最好的操作） |
| 卖出三条件之一触发 | **卖出** |

## 第六部分：心态守则

### 5 个核心心态测试（买入前 + 持有中反复用）

1. **跌 50% 测试**：跌一半你会加仓吗？答 No 就不要买
2. **关盘 3 年测试**：交易所关 3 年你愿意持有吗？
3. **门外汉测试**：5 分钟讲清楚为啥买这家公司？
4. **报纸头条测试**：明天报纸说 X 行业要被监管，你会慌吗？
5. **十年视角测试**：10 年后这家公司大概率会变得更好 / 持平 / 更差？

### 反人性的纪律
- **牛市保持冷静**：不追高、不加杠杆、不开仓 X 个新仓位
- **熊市保持理性**：不割肉、不抛持仓、关注好公司打折
- **看新闻不要"动手"**：情绪化操作 = 反向指标
- **每天关注公司，不每天关注股价**
- **三个月不看账户也可以**（说明你信你的判断）

### "买入即合伙人"心态
- 不会因为合伙人某天表现差就开除他
- 不会每周问"换个合伙人怎么样"
- 但会持续观察他的诚信、能力、努力是否还在

## 第七部分：A 股 / 港股特别考量

### 中国市场的独特之处

#### A 股特点
- 散户多 + 波动大 → 错杀机会多，但陷阱也多
- 政策敏感 → 行业政策风险（医药集采、互联网监管、教培）
- T+1 + 涨跌停 → 流动性受限
- 上市结构限制 → 部分行业代表性不足（互联网龙头全在港美股）

#### 港股特点
- 估值长期低于 A 股 → 同一公司 H 股折价
- 流动性较 A 股差 → 注意成交量
- 机构主导 → 价值发现相对慢
- AH 折溢价：可作参考但不要纯套利
- ⚠️ 老千股高发地带（市值小 / 流动性差 / 实控人差）→ **一律不碰**

### 行业政策风险清单（A 股 / 港股共有）
- 医药集采 → 影响仿制药、部分器械
- 互联网反垄断 → 影响平台公司
- 教培监管 → 历史归零案例
- 房地产 → 地产链全面承压
- 国企改革 → 国企估值修复机会
- 双碳目标 → 能源 / 化工 / 钢铁长期影响

### 国企 vs 民企的投资风格匹配
- **国企**：看政策周期 + CEO 任期 + 分红稳定 → 适合"低估值高股息"风格
- **民企**：看老板 + 治理 + 增长 → 适合"成长 + 护城河"风格

### 港股投资特殊提醒
- 印花税 + 汇率成本 → 长持更划算
- AH 折溢价 > 30% 可作参考（不是直接套利信号）
- 港股通标的优先（流动性 + 合规）
- 关注大股东持股变化（H 股披露更详细）

## 附录 C：财务红线的动态调整原则

财务红线不是死板数字，必须按**行业 / 公司阶段 / 市场环境**调整。

**核心原则**：任何"放宽"都要有明确理由（写在评估报告里）。不能因为"我喜欢这家公司"就放宽。同一时间、同一公司，用同一套调整后的标准。

### C.1 按行业差异化

| 行业 | 不适用的红线 | 改看 |
|---|---|---|
| **金融**（银行 / 保险 / 券商） | 资产负债率 < 60%（银行天然高杠杆 90%+） | 资本充足率 / 不良率 / 拨备覆盖率 / 净息差；ROE > 12% 可接受 |
| **周期股**（钢铁 / 煤炭 / 化工 / 有色 / 海运） | 单年 ROE > 15% 不必苛求 | PB 历史低位 + 高股息 + 龙头成本曲线；行业景气底部最佳 |
| **重资产成长期**（光伏 / 锂电 / 半导体设备） | FCF / 净利润 > 70% 暂不适用 | 经营性现金流 + 扩产节奏合理性 + ROIC 趋势 |
| **创新药 / 早期 SaaS** | ROE / 净利润标准 | 管线进度 + 现金储备撑几年 + 单位经济；警惕"永远明年盈利" |
| **互联网 / 平台型** | 资产负债率压力天然小 | FCF + 货币化率 + 用户增长 + 单位经济；警惕流量见顶仍突破不了货币化 |

### C.2 按公司生命周期阶段

| 阶段 | 红线宽严 | 关注重点 |
|---|---|---|
| **早期成长**（增长 > 30%） | 财务红线适度放宽 | 商业模式 + 单位经济 + 现金消耗速度 |
| **稳定成长**（10-30%） | **严格执行全部红线** | 全部 6 维度 |
| **成熟期**（< 10%） | 红线 + 分红 / 回购权重提升 | 估值 + 股东回报 |
| **衰退期** | 一般不投资 | 除非"价值修复"特殊机会（且要极强能力圈） |

### C.3 按市场环境

- **低估值 + 高利率市场**（如 2022-2024 部分时段）：估值红线**严格**执行 — 市场给折扣，挑剔一点；FCF/EV 对比国债更有意义
- **高估值 + 低利率市场**：估值红线适度放宽，但**商业模式 + 护城河红线不能放宽**
- 段永平："好生意贵一点也可以分批买；烂生意便宜也不能碰"

### C.4 按时点 vs 趋势

- **单季 / 单年异常** ≠ 结构性问题
- 看**连续 4 季度走势** > 单季数据
- 看 **3-5 年趋势** > 短期波动
- 例：单季 ROE 跌破 15% 不必立即卖，连续 4 季都跌破要查根本原因

### C.5 调整时必须遵守的纪律

1. 任何"红线放宽"都必须**明确说明理由**（行业特性 / 公司阶段 / 市场环境）
2. 不能因为"我喜欢这家公司"就放宽
3. 同一公司同一时间，用同一套调整后的标准
4. 每份评估报告里**写出**调整了哪些指标、为什么调整
5. 红线**收紧**不需要理由（更保守永远 OK）；红线**放宽**必须举证

> **Claude 在为 Leslie 做每只股票评估时，必须遵守 C.1-C.5 的调整逻辑。**
> 若调整原则间冲突（如：金融业 + 早期成长），按"**更保守的那个**"执行。
